## 一、用途分流规则

### A. 封面类（X封面 / PPT封面 / 商业报告封面 / 作品集封面 / 公众号封面）

封面不是内容页，只需要：
- 一个巨型主视觉词
- 一个核心商业隐喻
- 一个完整但克制的小标题系统
- 极少量标签和一句短结论
- 少量细线、编号、网格、节点

信息密度要求：信息量低、留白充足、视觉中心明确、字体粗细克制。

**封面严禁**：5步流程图、完整漏斗图、复杂矩阵分析、大量图标、大段说明文字、过多模块、过粗字体。

### B. 信息图类（信息图 / 咨询分析页 / 方法论图 / 流程图 / 矩阵图）

信息图允许：
- 3–6 个主要信息模块，1 个核心结构图，1 条主叙事路径
- 少量数据点、标签、编号、图例
- 流程、矩阵、漏斗、路径、系统架构、价值链
- 清晰的标题、副标题、注释和结论区

**信息图严禁**：图标堆砌、装饰性插画、颜色过多、随机模块拼贴、大段说明。

## 二、长标题自动提炼

当主题较长时，自动拆为三层：

- **A 层 — 主视觉核心词**：最有冲击力的 2–8 字（中）/ 1–4 词（英），可被视觉重构
- **B 层 — 完整标题**：保留完整主题，放顶部/边缘信息栏，字号较小
- **C 层 — 极简标签**：如 STRATEGY / MARKET INSIGHT / BUSINESS MODEL，精确克制

---

## 三、标题信息重构原则

主视觉文字必须经过设计级重构，使文字本身承载主题隐喻。

**语义重构映射**：
- 增长 → 扩张、上升、飞轮、复利结构
- 转化 → 收敛、筛选、漏斗、路径结构
- 战略 → 路径、坐标、分叉、决策结构
- 系统 → 模块、链路、节点、架构结构
- 风险 → 阈值、防线、断层、分级结构
- 市场 → 坐标、窗口、区块、竞争格局
- 价值 → 阶梯、刻度、容器、交换结构
- 效率 → 压缩、加速、流动、自动化结构
- 机会 → 开口、窗口、突破点、增长通道
- 组织 → 层级、重组、协作网络、治理结构

**图文融合方式**：商业图形穿过字形、路径线进入文字内部、漏斗藏在负空间、矩阵网格成字骨架、阶梯做字底座、数据流穿字而过。

## 四、视觉基因

### 配色
主色：白色、深蓝 `#003A70`、黑色、冷灰、浅灰。
强调色（仅限关键节点）：克制蓝、极少量橙/绿/红。
**禁止**：蓝紫霓虹、赛博朋克光效、花哨渐变、彩虹色、高饱和广告色、大面积暖色背景。

### 字体
现代、清晰、克制、稳定。中文用黑体/宋黑混合，英文用 Helvetica / Univers 气质。
封面字体可大但粗细克制，可半镂空线重构。信息图层级明确，正文小字简短。

### 质感
可使用：轻微纸张纹理、极淡阴影、精准细线、极简网格、细边框、小型编号系统、页眉页脚。
必须保持：干净、克制、稳定、专业、高级、理性。

## 六、构图规则

### 封面构图
1. 顶部信息栏：完整标题、标签、年份
2. 中央主视觉区：巨型重构文字 + 核心隐喻
3. 边缘锚点：起点词/终点词/价值词，最多 2–3 个
4. 底部短结论：12–22 字以内

### 信息图构图
1. 顶部标题区：完整标题 + 副标题 + 标签
2. 中央结构图区：核心流程/矩阵/系统/路径
3. 左右辅助模块：关键洞察、变量、约束
4. 底部结论区：1–3 条关键结论
5. 图例与编号：极小字号、细线、浅灰

阅读路径：左→右、上→下、问题→方案、输入→输出。

---

## 七、严格禁止

- 普通大标题、没有文字重构、没有视觉隐喻
- 视觉隐喻和文字割裂
- 普通 PPT 模板感
- 商务小人、握手图标、卡通人物、机器人头像
- 科幻蓝紫背景、花哨渐变、彩虹色
- 复杂到失控的信息图、内容无层级
- 字体过粗或过花、画面过满
- 自媒体廉价感

---

## 八、最终标准

1. 视觉上像顶级咨询公司作品
2. 主题表达清晰、标题经过信息重构
3. 图形和文字融合、麦肯锡式克制
4. 可直接用于 PPT 封面 / X 封面 / 商业报告封面
5. 即使去掉大部分装饰，核心结构仍然成立
