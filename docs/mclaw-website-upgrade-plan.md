# MClaw 官网升级开发计划

> 2026-06-15 · 从通用 SkillHub 升级为垂直地铁资源经营 AI 平台官网

---

## 1. 背景

当前 `apps/web/` 是一个通用 "mclaw SkillHub" 技能市场网站，定位是 AI Skills 推荐与分发。现在需要升级为 **MClaw —— 专为地铁资源经营打造的垂直 AI 平台**官网，增加产品介绍、专家库、行业场景、系统连接等核心板块，从通用技能市场转型为行业垂直 AI 产品官网。

参考竞品：QClaw（腾讯管家）、ZCode、CodeBuddy 的信息架构和 CTA 模式。

## 4. 首页区块设计（核心改动）

按内容草稿从上到下，8个区块：

### §1 Hero（品牌+下载CTA）

- 产品标题：**MClaw｜专为地铁资源经营打造的 AI 平台**
- 副标题：整合行业知识、经营方法、地铁数据、车站画像、专业技能与专家服务，帮助用户更高效地完成行业研究、经营分析、体系设计、方案生成和业务决策。
- 补充说明：支持独立使用，也可连接 Union3.0 等数字化经营系统，进一步基于企业真实数据开展深度分析和业务协同。
- 双下载 CTA：**下载 Mac 版** / **下载 Windows 版**
- 辅助链接：查看技能热榜

### §2 价值主张（不是通用AI）

- 标题：**不是通用 AI，而是更懂地铁资源经营**
- 说明：通用 AI 可以回答问题，但不一定理解地铁资源经营的业务场景。
- MClaw 围绕地铁行业的真实经营问题进行设计，覆盖地铁商业、媒体广告、资源管理、车站画像、非票务收入、生活圈运营、招商经营、行业研究和经营体系设计等方向。
- 底部强调：它服务的是地铁资源经营行业的从业者，而不是泛办公、泛娱乐或通用问答场景。

### §3 独立使用（不连接系统也能使用）

- 标题：**不连接系统，也能使用**
- 副标题：行业知识、数据分析和经营设计，开箱即用
- 说明：即使不连接企业数字化经营系统，MClaw 也可以作为地铁资源经营行业的 AI 平台使用。
- 6条能力要点：
  1. 查询地铁资源经营、非票务收入、车站商业、媒体经营等行业知识
  2. 学习国内外地铁商业、资源经营和生活圈运营的先进理念
  3. 分析全国地铁城市、线路、站点、客流和商业发展情况
  4. 生成车站画像、商圈分析、资源价值判断和经营建议
  5. 辅助设计经营架构、业务流程、指标体系和产品方案
  6. 生成行业研究报告、经营分析报告、招商材料和汇报方案

### §4 系统连接（连接系统后更深入）

- 标题：**连接系统后，更深入**
- 副标题：连接数字化经营系统，释放企业数据价值
- 说明：对于已经建设 Union3.0 或其他数字化经营系统的客户，MClaw 可以进一步连接企业业务数据，让 AI 基于真实经营数据开展分析和输出。
- 6个可连接方向卡片：
  1. Union3.0 数字化经营系统
  2. 地铁生活圈
  3. 媒体商城
  4. 站点画像系统
  5. 商业资源系统
  6. 经营数据系统
- 提示：系统连接不是使用 MClaw 的前提，而是企业客户的增强能力。

### §5 技能与专家

- 标题：**把行业能力做成技能和专家服务**
- 说明：MClaw 将地铁资源经营中的行业知识、经营方法、数据分析能力和专家经验，沉淀为可直接使用的技能与专家服务。用户可以通过专家获得专业判断，也可以通过技能快速完成具体任务。
- 3个CTA：**查看专家** / **查看技能热榜** / **浏览全部技能**

### §6 适合谁使用

- 标题：**适合谁使用**
- 说明：MClaw 面向地铁资源经营行业的相关人员和机构，帮助不同角色提升行业认知、分析判断、方案设计和业务执行效率。
- 6个角色卡片：

| 角色 | 描述 |
|------|------|
| 地铁业主单位 | 用于学习先进经营理念，设计资源经营体系，分析资源价值，辅助经营决策和管理汇报。 |
| 媒体经营公司 | 用于分析媒体资源价值，设计广告产品，生成客户方案，辅助投放建议和经营复盘。 |
| 商业招商团队 | 用于学习业态规划方法，辅助招商定位、品牌匹配、租金判断和招商材料生成。 |
| 生活圈运营团队 | 用于获取运营方法、活动策划、商户联动、用户分析和复盘建议。 |
| 行业研究人员 | 用于查询全国地铁数据、行业趋势、标杆案例、先进经营理念和城市对比分析。 |
| 经营管理层 | 用于快速获得经营架构、分析结论、专题报告和管理汇报材料。 |

### §7 开始使用 CTA

- 标题：**开始使用 MClaw**
- 说明：当前支持 Mac 和 Windows 桌面端。
- 下载按钮：**下载 Mac 版** / **下载 Windows 版**
