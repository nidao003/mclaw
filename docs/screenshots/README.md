# 开发参考截图

本目录存放开发过程中的**参考素材和效果截图**，仅本地留存，**不纳入 git 跟踪**（见根目录 `.gitignore`）。

## 分类

| 目录 | 说明 |
|------|------|
| `codebuddy/` | CodeBuddy 官网截图（hero / pricing / features / FAQ 等） |
| `qclaw/` | QClaw 截图（hero / nav / pricing / demos 等） |
| `oohforce/` | OOHForce 截图（about / product / solution 等） |
| `nexu-web/` | Nexu Web 截图（home / models / skills 等） |
| `mclaw/` | mclaw 自家页面截图（hero / home / homepage-upgrade 等） |
| `misc/` | 通用截图（pricing / features / testimonials / scenarios 等） |

## 说明

- 这些图是设计阶段的竞品参考与自家页面效果记录，不属于源码资源。
- 应用图标、应用截图、社区二维码等**正式资源**位于 `resources/` 和 `src/assets/`，正常纳入 git。
- 如需新增截图，直接放入对应分类目录即可，不会被 git 跟踪。
